# 智慧購物清單 (Smart Shopping List)

一個幫助您進行理性消費決策的網頁應用。

## 核心特性

✨ **自動排序**：「需要」的物品永遠優先於「想要」的物品，同類別中低價優先

💾 **本地儲存**：所有資料儲存在您的瀏覽器中，無需登錄

📱 **響應式設計**：支援桌面、平板和手機

🎨 **簡潔界面**：現代、乾淨的設計風格

## 使用方法

1. 開啟 [智慧購物清單](https://yourusername.github.io/smart-shopping-list/)
2. 在輸入頁面填寫物品名稱、價格和分類（需要/想要）
3. 點擊「新增物品」按鈕
4. 點擊「查看清單」查看自動排序後的結果
5. 在清單頁面可以刪除物品或清空整個清單

## 排序邏輯

清單會根據以下規則自動排序：

| 優先級 | 規則 |
|--------|------|
| 1 | 「需要 (Need)」優先於「想要 (Want)」 |
| 2 | 同類別中，價格低的優先 |

## 技術棧

- HTML5
- CSS3

- Chrome
- Firefox

## 隱私

所有資料都儲存在您的瀏覽器本地，不會上傳到任何伺服器。
